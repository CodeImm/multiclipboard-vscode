# Changelog

All notable changes to the MultiClipboard extension are documented in this file.

## [1.0.0] - 2025-12-01

### Added

- Initial release
- Automatic clipboard monitoring
- Quick Pick interface for clipboard history
- Keyboard shortcut (Ctrl+Shift+V / Cmd+Shift+V)
- Persistent storage (up to 100 items)
- Timestamp display with smart formatting
- Search and filter functionality
